## Cardo 簡介
- 一個將名片 e 化的 APP
- 使用者可以用 APP 來製作自己的名片
- 透過 QR code 掃描可以下載別人的名片
- 查看名片時可以快速開啟連結（例如 Facebook 個人檔案）

![](cardo-demo-image.png)

## Cardo 已實現功能
- 編輯、新增和刪除自己的 Cardo 
- 編輯模式下可加入文字方塊並拖曳、更改文字顏色和大小、加入連結
- 儲存、瀏覽自己和別人的 Cardo ，並可產生 QR code
- 透過 App 內的相機掃描 QR code 下載別人的名片
- 查看名片時可以快速開啟連結（例如 Facebook 個人檔案）
